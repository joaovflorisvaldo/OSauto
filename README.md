# Sistema de Controle de Ordens de Serviço (OS) para Mecânica de Carros

Este é um sistema simples de controle de Ordens de Serviço (OS) para uma mecânica de carros, desenvolvido em Java com interface gráfica usando Java Swing.

## Funcionalidades

- Cadastro de clientes
- Cadastro de veículos
- Abertura de ordens de serviço

## Tecnologias Utilizadas

- Linguagem de programação: Java
- Interface gráfica: JFrame

## Como Executar

1. Certifique-se de ter o Java Development Kit (JDK) instalado em sua máquina.
2. Clone este repositório ou faça o download dos arquivos.
3. Navegue até o diretório do projeto no terminal.
